# Cotizador de Insumos

Proyecto listo para subir a Vercel.

## Ejecutar localmente

```bash
npm install
npm run dev
```

## Publicar en Vercel

1. Subí esta carpeta a un repositorio de GitHub.
2. Entrá a Vercel.
3. Elegí **Add New Project**.
4. Importá el repositorio.
5. Vercel detecta Vite automáticamente.
6. Hacé click en **Deploy**.

## Build

```bash
npm run build
```

## Notas

- Para que Google Sheets funcione, usá un link público y con la pestaña correcta (`gid`).
- Si la hoja tiene los artículos en la columna A, cada fila se toma como un producto.
